# 开放应用

## 使用

在需要加载模块应用的配置文件中添加

```yaml title="alemon.config.yaml"
apps:
  - 'alemonjs-name'
  - 'alemonjs-name2'
  - 'alemonjs-name3'
```

> 需要重启
